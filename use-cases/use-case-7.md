USE CASE: 7 View an Employee’s Details for a Promotion Request
CHARACTERISTIC INFORMATION
Goal in Context

As an HR advisor I want to view an employee’s details so that the employee’s promotion request can be supported.

Scope

Company.

Level

Primary task.

Preconditions

Employee record and promotion-related information exist.

Success End Condition

HR has the employee details needed to process a promotion request.

Failed End Condition

Information is incomplete or unavailable.

Primary Actor

HR Advisor.

Trigger

An employee submits a promotion request or manager requests HR support.

MAIN SUCCESS SCENARIO

Promotion request is received.

HR searches for and opens the employee’s record.

HR reviews current title, salary, performance notes (if available) and manager info.

HR records decision / forwards information to promotion panel.

EXTENSIONS

Employee record incomplete:

HR requests additional documentation from the manager/employee.

SUB-VARIATIONS

None.

SCHEDULE

DUE DATE: Release 1.0
